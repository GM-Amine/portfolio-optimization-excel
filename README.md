# Portfolio Optimization and Performance Analysis (Excel)

## Description
This project focuses on investment portfolio optimization and performance analysis using Microsoft Excel only.  
The objective is to evaluate the riskâ€“return profile of a portfolio composed of selected U.S. equities and to analyze its overall performance.

## Assets Analyzed
The portfolio is composed of the following companies:
- Waters Corporation
- DR Horton
- Interpublic Group (IPG)
- Lockheed Martin

## Methodology
The analysis was conducted entirely in Excel and includes:
- Computation of asset returns
- Descriptive statistics
- Risk analysis (volatility)
- Portfolio performance evaluation
- Comparative analysis of individual assets and the overall portfolio

All calculations, tables, and visualizations were created using built-in Excel functionalities.
